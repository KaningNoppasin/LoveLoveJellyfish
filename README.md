## Setup
**Activate the virtual environment**:
- **On Windows**:
```bash
cd pygame
python -m venv pygame_env
.\pygame_env\Scripts\activate
```

- **On macOS and Linux**:
```bash
cd pygame
python3 -m venv pygame_env
source pygame_env/bin/activate
```

**Install Required Libraries**:
```bash
pip install pygame
pip install numpy
pip install pyaudio
pip install pyserial

```

## Running the Project
```bash
python src/main.py
```
